== Description ==

ONTRApages for WordPress allows ONTRAPORT & ONTRApages Premium users to connect to their accounts and easily host their landing pages on their own WordPress sites.

== Installation ==

1. Upload and unzip the contents of ONTRApages.zip to the `/wp-content/plugins/ontrapages` directory or get it by searching the WP plugin repository.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Do I need an ONTRAPORT account? =

You need an ONTRAPORT or ONTRApages Premium account to generate the API credentials for this plugin.

= Will you ever support ONTRApages.com accounts? =

You can use this plugin with ONTRApages Premium accounts now! You can try ONTRApages for free at [https://ontrapages.com](https://ontrapages.com) and then upgrade to ONTRApages Premium for $15 / month or $99 / year to gain more functionality, including using this plugin.

= Where can I learn more about ONTRApages? =

Visit ONTRApages.com for more information about ONTRApages.

= Anywhere I can go to learn more or get support information? =

Visit our Facebook Support Group at [https://www.facebook.com/groups/ONTRApagesusercommunity/](https://www.facebook.com/groups/ONTRApagesusercommunity/) Or, open a free ONTRApages account at [https://ontrapages.com](https://ontrapages.com) to test the page building functionality before upgrading to ONTRApages Premium ($15 / month or $99 / year) for the extra features (including using this plugin to publish your pages to WordPress).
